# Gemini AI Clone - AI Chat Application

A fully-functional AI chat application built with React.js that mimics Google's Gemini AI interface. Features real-time conversations with typing effects, chat history, and responsive design.

## Features

### AI Chat Functionality
- Real-time conversations with Google's Gemini AI
- Typing effect animations for AI responses
- Markdown support with bold text formatting
- Loading animations during API calls

### User Interface
- Responsive design (mobile, tablet, desktop)
- Collapsible sidebar with chat history
- Dark theme with gradient accents
- Smooth animations and transitions
- Interactive cards with hover effects

### User Experience
- Chat history persistence
- Quick prompts from cards
- New chat creation
- Real-time input validation
- Loading states with skeleton loaders

### Technical Features
- React Context API for state management
- Custom hooks for API integration
- Error handling and retry mechanisms
- Performance optimized with React.memo
- CSS animations with keyframes

## Tech Stack

### Frontend
- **React.js** (Hooks, Context API)
- **CSS3** (Flexbox, Grid, Animations)
- **JavaScript** (ES6+)

### API Integration
- **Google Gemini AI API**
- **REST API calls**

### Development Tools
- **Vite** (Build tool)
- **Git** (Version control)
- **ESLint** (Code quality)
